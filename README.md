# Tender-Hack
Cервис группового подбора стандартных товарных единиц (СТЕ) по их характеристикам.

Цель- Разработка сервиса группового подбора стандартных товарных единиц (СТЕ) по их характеристикам.
Что нужно сделать
Участникам предстоит:
- проанализировать предоставленные данные;
- определить значимые характеристики товаров для агрегации;
- реализовать механизм группировки СТЕ по релевантным признакам;
- добавить функции:
  1. выдача агрегированных СТЕ по запросу пользователя;
  2. редактирование и управление связями товаров;
  3. оценка качества агрегации;
  4. сохранение и перегенерация результатов;
- создать **удобный веб-интерфейс**;
- провести тестирование и представить проект на защите.
